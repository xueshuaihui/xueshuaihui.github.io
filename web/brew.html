# 更新 Homebrew

brew update

# 安装包

brew install appName

# GUI

# Cakebrew

Cakebrew 是 Homebrew 的 GUI 管理器，在 Cakebrew 中，你可以看到当前所有已经安装的软件，并可以在 Caskbrew 中对其他软件执行升级等操作。

# install

brew install --cask cakebrew
